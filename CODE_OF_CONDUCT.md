# Code de conduite

## Notre engagement

Ce projet est un espace bienveillant. Toute contribution, quel que soit le niveau d'expérience, est la bienvenue.

## Comportements attendus

- Être respectueux et constructif dans les échanges
- Accepter les retours avec ouverture
- Se concentrer sur ce qui est le mieux pour le projet

## Comportements inacceptables

- Insultes, harcèlement ou attaques personnelles
- Publication d'informations privées sans consentement
- Toute conduite qui serait jugée inappropriée dans un cadre professionnel

## Application

Les comportements inacceptables peuvent être signalés à : damien.paszkiewicz@live.fr

Les mainteneurs se réservent le droit de supprimer des commentaires ou de bannir des contributeurs qui ne respectent pas ce code de conduite.
