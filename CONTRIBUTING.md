# CONTRIBUTING.md

## 🚀 Comment contribuer
1. Forkez le dépôt
2. Clonez votre fork
3. Créez une branche
4. Faites vos modifications
5. Testez votre code
6. Soumettez une Pull Request

## 🧠 Règles de style
- Utilisez Prettier pour formater le code
- Respectez les conventions ESLint

## 🐞 Signaler un bug
Ouvrez une issue avec un titre descriptif et les étapes pour reproduire.

## 💬 Besoin d’aide ?
Contactez l’équipe sur le salon Discord du projet.
